# vue-all

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### 安装vue-router
```
yarn add vue-router
```
+ main.js文件中
```
import VueRouter from 'vue-router'  // 引入vue-router

Vue.use(VueRouter)  // 声明使用
```
